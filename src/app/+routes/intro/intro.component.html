Goal:

The interaction and motion design we're explicitly kept simple, only using Material Design and Google Photos as design precedence.

Motion:

Material design's motion is meticulously justified Borrowing from those justifications I tried to achieve motion in the UI that was:

  • Responsive
  • Natural
  • Aware
  • Intentional

By making the motion: Quick, clear, and cohesive

The only curve and easing use for entering items:

export const STANDARD_EASE = '250ms cubic-bezier(.35, 0, .25, 1)';

The only curve and easing used for leaving items:

export const STANDARD_LEAVE = '195ms cubic-bezier(.35, 0, .25, 1)';
